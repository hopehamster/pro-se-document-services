# Deployment Instructions

The website prototype has been successfully built and pushed to your GitHub repository:
**[https://github.com/hopehamster/pro-se-document-services](https://github.com/hopehamster/pro-se-document-services)**

## To Enable GitHub Pages:
1. Go to the **Settings** tab of the repository.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment**, select **Source** as "Deploy from a branch".
4. Under **Branch**, select **main** and folder **/(root)**.
5. Click **Save**.

Your site will be live at:
**https://hopehamster.github.io/pro-se-document-services/**
